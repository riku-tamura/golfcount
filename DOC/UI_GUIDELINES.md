# UI Guidelines

## 操作性の方針

Apple Watch では、指先での即時操作と短時間での視認性を優先しています。

今回の UI 方針:

- 数字を大きく表示する
- `+1 / -1` ボタンを大きいタップ領域にする
- 各カードを独立させて、押し間違いを減らす
- 色で増減の意味をすぐ判断できるようにする
- スクロール時も全体状況が先頭で把握できるようにする
- 合計スコアとホール選択を先頭で完結させる

## 実装した改善点

- ボタン高を大きめに固定
- 値表示部分を別パネル化して視認性を上げた
- 長押しで連続変更できるよう `buttonRepeatBehavior(.enabled)` を使用
- 18ホール合計を最上部に大きく表示
- 現在ホールは `1H〜18H` から直接選べるようにした
- 増減とホール選択に触覚フィードバックを入れた
- VoiceOver 向けに各ボタンの操作内容を明示した
